# Roadmap

## Current MVP

- Local Git repository summary
- Linux subsystem learning path
- Repository-relative source viewer
- Per-file status and notes
- Local JSON persistence

## Next: code navigation

- Directory tree API and UI
- Symbol search with `git grep`
- Back/forward navigation history
- Recent files and bookmarks
- Line-linked notes

## Next: Git workflow

- Commit history for the current file
- `git blame` for a selected line range
- Diff viewer for working-tree changes
- Branch selector in read-only mode
- GitHub remote URL normalization and commit links

## Later: learning system

- Custom learning paths stored as JSON
- Daily reading queue
- Review questions per file
- Export and import notes
- Multiple repository profiles

## Before network deployment

- Authentication
- Strict repository allowlist
- CSRF protection
- Narrow CORS policy
- Audit logging
- Rate limits
